datatypes of Javascript

datatype is used to define type of value.

There are different types of datatypes in js

String : String is nothing but sequence of characters

Number : These are numbers eg. 1,8,0,199

Boolean : These are boolean values eg. True and false

Undefined : represents undefined values

null : represent null value ie. no value.. not even 0

Object : It is the most important data-type and forms the building blocks for modern JavaScript.
